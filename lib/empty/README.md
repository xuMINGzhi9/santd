## API

| 参数        | 说明                                           | 类型     | 默认值                          |
| ---         | ---                                            | ---      | ---                             |
| description | 自定义描述内容                                 | string \| slot \| bool | - |
| imageStyle  | 图片样式                                       | string   | -                               |
| image       | 设置显示图片，为 string 时表示自定义图片地址。 | string   | `Empty.PRESENTED_IMAGE_DEFAULT` |
