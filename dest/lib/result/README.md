## API

| 参数     | 说明                                                          | 类型          | 默认值    |
| ---      | ---                                                           | ---           | ---       |
| title    | title 文字                                                    | string        | -         |
| subTitle | subTitle 文字                                                 | string        | -         |
| status   | 结果的状态,决定图标和颜色                                     | 'success' \| 'error' \| 'info' \| 'warning'\| '404' \| '403' \| '500' | 'info' |  |
| icon      | 自定义icon                                          | slot        | -         |
| extra   | 操作区                                  | slot        | -         |
